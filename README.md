# CapAI

一个致力于多模态低成本AI的项目。

## 多模型交互

针对GPT等现有AI的缺陷，CapAI项目将考虑采用多模型交互的方式运行，使用不同AI模型处理不同领域的任务，由一个相对强大的语言模型进行统筹调配。

## 现有模型
